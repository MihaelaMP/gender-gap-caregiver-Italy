# gender-gap-caregiver-Italy
Analisi del divario di genere nell'inattività per motivi familiari in Italia (dati ISTAT, Power BI)

📊 Donne e inattività per motivi familiari in Italia

🎯 Obiettivo
Analizzare il ruolo delle responsabilità familiari nell’inattività lavorativa e il loro contributo al divario di genere in Italia.

📂 Fonte dati
 - ISTAT — Rilevazione Forze di Lavoro
 - Periodo: 2024–2025
 - Popolazione: 15–64 anni

🛠️ Strumenti
 - Power BI Desktop
 - Power Query
 - DAX

🔍 Metodologia
- Integrazione di dataset separati per sesso (maschi, femmine, totale)
- Pulizia e normalizzazione dei dati
- Filtro su inattivi per motivi familiari
- Analisi comparativa per genere
- Analisi del trend temporale

📈 Risultati principali
Le donne risultano significativamente più inattive per motivi familiari rispetto agli uomini
Nel 2025:
- circa 3,1 milioni di donne inattive per motivi familiari
- contro 0,2 milioni di uomini
Il rapporto donne/uomini è pari a 20,5x.
Il fenomeno mostra un aumento nel periodo 2024–2025.

📊 Visualizzazioni

1. Confronto per genere (2025)
  → le donne risultano nettamente più inattive per motivi familiari
2. Trend temporale (2024–2025)
  → crescita del numero di donne inattive
3. Indicatore sintetico
  → rapporto donne/uomini ≈ 20,5

💡 Insight

I dati mostrano che le responsabilità familiari rappresentano una componente rilevante dell’inattività e risultano distribuite in modo fortemente squilibrato tra uomini e donne.

🔗 Collegamento agli altri progetti

Questo progetto si inserisce in una serie di analisi sul divario di genere nel mercato del lavoro:

- Analisi ISTAT (Python): relazione tra numero di figli e occupazione femminile
- Analisi Eurostat (Power BI): motherhood penalty
- Questo progetto: ruolo delle responsabilità familiari nell’inattività

📌 Conclusione

Le evidenze suggeriscono che il lavoro di cura rappresenta un fattore rilevante nella partecipazione femminile al mercato del lavoro, con possibili implicazioni sulla continuità lavorativa.

👉 Tre fonti diverse, risultati coerenti.
